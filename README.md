# EPG projektas IPTV grotuvams

Ši saugykla skirta IPTV grotuvams, naudojantiems EPG (elektroninio programų gido) duomenis.

## ✅ Veikianti EPG nuoroda

- Lengvesnė versija (~2000 kanalų):  
  `https://github.com/giedrius73/EPG/raw/main/epg-lite.xml.gz`

## 📌 Naudojimas

1. Įklijuokite nuorodą į savo IPTV grotuvo `EPG URL` laukelį.
2. Ši versija tinka silpnesniems grotuvams arba greitesniam įkėlimui.
3. Kai kuriems kanalams EPG gali būti nepilnas arba netikslus.

### ❗ Kaip pakeisti GitHub nuorodą, kad ji veiktų IPTV grotuvuose

Jeigu nuoroda į EPG failą atrodo taip:

https://github.com/giedrius73/EPG/blob/main/epg-lite.xml.gz

Ji **neveiks IPTV grotuvuose**, nes tai yra GitHub puslapio nuoroda, o ne tiesioginis failas.

✅ **Sprendimas:** pakeisk žodį `blob` į `raw`:

https://github.com/giedrius73/EPG/raw/main/epg-lite.xml.gz

Ši nuoroda veikia IPTV grotuvuose, nes ji pateikia failą tiesiogiai.

# EPG project for IPTV players

This repository provides EPG (Electronic Program Guide) data for IPTV players.

## ✅ Working EPG link

- Lite version (~2000 channels):  
  `https://github.com/giedrius73/EPG/raw/main/epg-lite.xml.gz`

## 📌 Usage

1. Paste the link into your IPTV player's `EPG URL` field.
2. This version is suitable for lighter setups or faster loading.
3. Some channels may have incomplete or inaccurate EPG data.
### ❗ How to fix GitHub link for IPTV players

If your EPG link looks like this:

https://github.com/giedrius73/EPG/blob/main/epg-lite.xml.gz

It **won’t work in IPTV players**, because it points to a GitHub page, not the actual file.

✅ **Solution:** replace `blob` with `raw`:

https://github.com/giedrius73/EPG/raw/main/epg-lite.xml.gz

This link works in IPTV players because it delivers the file directly.
